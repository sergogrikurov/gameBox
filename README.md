## About the Project
GameBox is a small browser-based game built with **Vue 3**.
The project demonstrates clean frontend structure, responsive layout, and deployment to production using **Firebase Hosting**.

This project was created as a portfolio example and can be easily adapted for other interactive web applications.

---

## Features
- Built with **Vue 3** and **Vite**
- Responsive layout (desktop & mobile)
- Clean and simple UI
- Fast loading and optimized build
- Deployed to Firebase Hosting

---

## Tech Stack
- Vue.js 3
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3
- Firebase Hosting
